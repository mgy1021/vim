# 第十五天

## 目标 🎯

掌握窗口的管理

## 任务点

### 新建窗口

左右：Ctrl + w + v
上下：Ctrl + w + s

### 窗口切换

Ctrl + w + hjkl

在双窗口模式下，窗口互换切换：Ctrl + w + w

### 关闭窗口

Ctrl + w + c
与 vsCode 的快捷键 Command + w 的效果是一致的，所以更加推荐使用 Command + w

注意：当窗口存在多个文件时，只会关闭当前打开的文件，当前窗口并不会关闭
多个文件时关闭窗口：使用 vsCode 的快捷键 Command + k + w
只保留当前窗口，关闭其他所有窗口：Ctrl+w + o
